Данная программа предназначена для полуавтоматического снятия данных о записи на тренировку.
При запуске вводится дата, за которую запрашиваются данные.
В нечётные будние дни записи выводятся с 12.00 по 20.00, в чётные - с 10.00 по 17.00.
Личные договорённости о внеплановом посещении игнорируются в связи с их незадокументированностью.
Если на час записано более 8 человек, последние отсекаются и не вносятся в список.
